# AstrBot 插件：模拟修仙 v2

> **版本：** v3.1.61  
> **许可证：** AGPL-3.0  
> **作者：** xiaojuwa  
> **仓库：** <https://github.com/xiaojuwa/astrbot_plugin_monixiuxian2>

一款面向 AstrBot 的文字修仙插件，支持修炼、突破、装备、丹药、宗门、世界 Boss、秘境、历练、悬赏、洞天、灵田、双修、灵眼，以及配套的银行、黑市、悟道、福缘和 Web 预览界面。

---

## 功能概览

| 模块 | 说明 |
|---|---|
| 基础修仙 | 创建角色、查看信息、签到、改道号、弃道重修、逆天改命 |
| 修炼突破 | 闭关、出关、突破信息、突破丹、轻量天劫 |
| 装备丹药 | 装备/卸下、丹药背包、服药、物品信息 |
| 储物戒 | 自动入袋、搜索、赠予、丢弃、更换储物戒 |
| 商店经济 | 丹阁、器阁、百宝阁、银行、贷款、突破贷款、黑市 |
| 战斗玩法 | 切磋、决斗、世界 Boss、传承挑战 |
| 挂机玩法 | 历练、秘境、悬赏、炼丹 |
| 长线玩法 | 洞天、灵田、双修、灵眼、宗门 |
| 增量玩法 | 悟道、福缘 |
| Web 预览 | 只读页面，可查看角色、排行榜、Boss、秘境、洞天、灵田等信息 |

## 当前规则补充

| 系统 | 规则 |
|---|---|
| 世界 Boss | 战败后战斗 HP 降为 1，5 分钟后可再次挑战，战斗 HP 按时间自动恢复 |
| Boss 狂暴 | Boss 血量低于 30% 时进入狂暴，持续 5 回合并随机触发狂暴技能 |
| 黑市 | 分类定价与限购：普通丹 2 倍/5 颗，强化丹 2.5 倍/3 颗，突破丹 3 倍/2 颗，稀有丹 4 倍/1 颗 |
| 轻量天劫 | 金丹及以上突破成功后追加天劫判定，失败不会死亡，但会使本次突破作废并额外损失修为 |
| 储物戒 | 普通物品自动入戒，丹药自动进丹药背包；玩家侧不再支持手动存入与取出到普通背包 |
| 洞天置换 | 支持 `/置换洞天`，旧洞天按原价 60% 折算并重置收取时间 |
| 灵眼抢夺 | 有主灵眼需确认后决斗，败者 30 分钟内无法再次发起决斗 |

| 银行 | 已支持 |
| 洞天 | 已支持 |
| 历练 | 已支持 |
| 悬赏 | 已支持 |
| 灵田 | 已支持 |
| 灵眼 | 已支持 |
| 双修 | 已支持 |

### 启动方式

```bash
python web_preview_server.py --db D:\PycharmProjects\astrbot_plugin_monixiuxian2\xiuxian_data_lite.db
```

### Web 配置

编辑 `config/game_config.json`：

```json
{
  "web_server": {
    "host": "0.0.0.0",
    "port": 8765
  }
}
```
