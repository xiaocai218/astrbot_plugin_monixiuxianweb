# AstrBot 模拟修仙 v2

> 版本：`v3.1.84`  
> 许可证：`AGPL-3.0`  
> 作者：`xiaojuwa`

一个运行在 AstrBot 中的文字修仙插件，包含修炼、突破、装备、Boss、秘境、历练、宗门、洞天、灵田、灵眼、银行、黑市，以及配套的 Web 预览页面。

## 功能概览

| 模块 | 说明 |
|---|---|
| 基础修仙 | 创建角色、查看信息、签到、改道号、弃道重修、逆天改命 |
| 修炼突破 | 闭关、出关、突破、轻量天劫、悟道、福缘 |
| 装备与背包 | 装备、卸下、丹药背包、储物戒、搜索、赠予、丢弃 |
| 商店与银行 | 丹阁、器阁、百宝阁、银行、普通贷款、突破贷款、黑市 |
| 战斗玩法 | 切磋、决斗、世界 Boss、传承挑战 |
| 探索玩法 | 历练、秘境、悬赏、炼丹 |
| 长线玩法 | 宗门、洞天、灵田、双修、灵眼 |
| Web 预览 | 角色、排行榜、Boss、秘境、洞天、灵田、灵眼、银行等只读页面 |

## 常用指令

### 入门基础

| 指令 | 说明 |
|---|---|
| `我要修仙 [灵修/体修]` | 创建角色 |
| `我的信息` | 查看角色面板 |
| `签到` | 领取每日奖励 |
| `改道号 <名称>` | 修改道号 |
| `弃道重修 [确认]` | 删除角色并进入冷却 |
| `逆天改命` | 花费 10000 灵石重置灵根 |

### 修炼突破

| 指令 | 说明 |
|---|---|
| `闭关` | 开始修炼 |
| `出关` | 结算修为，并可能触发悟道 |
| `突破信息` | 查看突破要求、成功率与天劫提示 |
| `突破 [丹药名]` | 尝试突破，可使用破境丹 |
| `悟道信息` | 查看悟道说明 |
| `福缘信息` | 查看每日福缘次数 |
| `求福缘` | 主动领取一次福缘奖励 |

### 装备与背包

| 指令 | 说明 |
|---|---|
| `我的装备` | 查看当前装备 |
| `装备 <名称>` | 装备物品 |
| `卸下 <名称>` | 卸下装备 |
| `丹药背包` | 查看丹药库存 |
| `服用丹药 <名称>` | 使用丹药 |
| `储物戒` | 查看储物戒 |
| `丢弃 <物品> [数量]` | 删除储物戒中的物品 |
| `丢弃所有 <分类>` | 批量删除指定分类 |
| `搜索物品 <关键词>` | 搜索储物戒内容 |
| `更换储物戒 <名称>` | 更换更高级储物戒 |
| `赠予 <@某人> <物品> [数量]` | 赠送物品 |
| `接收 / 拒绝` | 处理赠予请求 |

### 商店与银行

| 指令 | 说明 |
|---|---|
| `丹阁 / 器阁 / 百宝阁` | 查看商店 |
| `购买 <物品名> [数量]` | 购买物品 |
| `物品信息 <名称>` | 查看道具详情 |
| `银行` | 查看存款、贷款和利息 |
| `存灵石 <数量>` | 存款 |
| `取灵石 <数量>` | 取款 |
| `领取利息` | 领取存款利息 |
| `贷款 <金额>` | 普通贷款 |
| `突破贷款 <金额>` | 突破专用贷款 |
| `还款` | 归还贷款 |
| `银行流水` | 查看银行流水 |
| `黑市` | 查看黑市丹药 |
| `黑市购买 <丹药名> [数量]` | 购买黑市丹药 |

### 战斗竞技

| 指令 | 说明 |
|---|---|
| `切磋 <@某人>` | 发起切磋 |
| `决斗 <@某人>` | 发起决斗 |
| `世界Boss` | 查看当前世界 Boss |
| `挑战Boss` | 挑战当前世界 Boss |
| `传承挑战 <@某人>` | 发起传承挑战 |
| `传承排行` | 查看传承排行榜 |

补充规则：
- 世界 Boss 每日可挑战 3 次，次日重置。
- Boss 战败后战斗 HP 会随时间恢复，`/我的信息` 可查看当前战斗 HP 与剩余次数。

## Web 预览

项目内提供独立的只读 Web 预览层，默认启动方式：

```bash
python web_preview_server.py --db xiuxian_data_lite.db
```

默认监听配置可在 `config/game_config.json` 中修改：

```json
{
  "web_server": {
    "host": "0.0.0.0",
    "port": 8765
  }
}
```

当前 Web 预览页已覆盖：
- 角色总览
- 排行榜
- 储物戒与背包
- 商店预览
- 世界 Boss
- 秘境
- 银行
- 宗门
- 悬赏
- 历练
- 洞天
- 灵田
- 灵眼
- 双修
