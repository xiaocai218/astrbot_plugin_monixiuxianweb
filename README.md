# AstrBot 模拟修仙 v2

> 版本：`v3.2.23`
> 许可证：`AGPL-3.0`  
> 作者：`xiaojuwa`

这是一个运行在 AstrBot 中的文字修仙插件，包含修炼、突破、装备、世界 Boss、秘境、历练、宗门、洞天、灵田、灵眼、灵宠、银行、黑市，以及配套的 Web 预览页面。

## 功能概览

| 模块 | 说明 |
|---|---|
| 基础修仙 | 创建角色、查看信息、签到、改道号、弃道重修、逆天改命 |
| 修炼突破 | 闭关、出关、突破、轻量天劫、悟道、福缘 |
| 装备与背包 | 装备、卸下、丹药背包、储物戒、搜索、赠予、丢弃 |
| 商店与银行 | 丹阁、器阁、百宝阁、银行、普通贷款、突破贷款、黑市 |
| 战斗玩法 | 切磋、决斗、世界 Boss、传承挑战 |
| 探索玩法 | 历练、秘境、悬赏、炼丹 |
| 长线玩法 | 宗门、洞天、灵田、双修、灵眼、灵宠 |
| Web 预览 | 角色、排行榜、Boss、秘境、洞天、灵田、灵眼、银行等只读页面 |

### 战斗竞技

| 指令 | 说明 |
|---|---|
| `切磋 <@某人>` | 直接开始切磋，用于战力评估 |
| `决斗 <@某人>` | 向对方发起决斗请求 |
| `接受决斗 / 拒绝决斗` | 处理收到的决斗请求 |
| `世界Boss` | 查看当前世界 Boss |
| `挑战Boss` | 挑战当前世界 Boss |
| `传承挑战 <@某人>` | 发起传承挑战 |
| `传承排行` | 查看传承排行 |

补充规则：
- 世界 Boss 每日可挑战 4 次，次日重置。
- 切磋可直接开始，仅用于战力评估，不消耗气血 / 灵气，也不附带正式惩罚。
- 决斗改为请求制：对方需要输入 `接受决斗` 后才会正式开打，未处理或拒绝都会取消本次请求。
- 论道为轻社交玩法：不触发战斗、不发放灵石奖励，双方获得少量修为并进入 1 小时冷却。
- Boss 战败后战斗 HP 会按时间恢复，`/我的信息` 可查看当前战斗 HP 与剩余次数。

## Web 预览

项目内提供独立的只读 Web 预览层，默认启动方式：

```bash
python web_preview_server.py --db xiuxian_data_lite.db
```

默认监听配置可在 `config/game_config.json` 中修改：

```json
{
  "web_server": {
    "host": "0.0.0.0",
    "port": 8765
  }
}
```

当前 Web 预览页已覆盖：
- 角色总览
- 排行榜
- 储物戒与背包
- 商店预览
- 世界 Boss
- 秘境
- 银行
- 宗门
- 悬赏
- 历练
- 洞天
- 灵田
- 灵眼
- 双修
