# AstrBot 模拟修仙 v2

> 版本：`v3.2.50`  
> 许可证：`AGPL-3.0`  
> 作者：`xiaojuwa`

这是一个运行在 AstrBot 中的文字修仙插件，包含修炼、突破、Boss、历练、宗门、洞天、灵田、灵眼、灵宠、银行与多种互动玩法，并提供配套的只读 Web 预览页面。

## 功能概览

| 模块 | 说明 |
|---|---|
| 基础修仙 | 创建角色、查看信息、签到、改道号、弃道重修、逆天改命 |
| 修炼突破 | 闭关、出关、突破、轻量天劫、悟道、福缘 |
| 装备背包 | 装备、丹药背包、储物戒、赠予、搜索 |
| 战斗玩法 | 切磋、决斗、世界 Boss、传承挑战 |
| 探索玩法 | 历练、秘境、悬赏、炼丹 |
| 长线玩法 | 宗门、洞天、灵田、双修、灵眼、灵宠 |
| 互动玩法 | 送灵石、论道、仙缘红包 |
| Web 预览 | 角色、排行榜、Boss、秘境、洞天、灵田、灵眼、银行等只读页面 |

## 当前规则摘要

| 模块 | 规则 |
|---|---|
| 世界 Boss | 每日可挑战 `4` 次，按玩家与日期重置 |
| 切磋 | 直接开始，仅用于战力评估，不消耗气血/灵气 |
| 决斗 | 采用请求制，对方接受后才会正式开打 |
| 论道 | 双方获得少量修为，并进入 `1 小时` 冷却 |
| 送灵石 | 单次最少 `100`，单次最多 `50000`，每日上限 `200000` |
| 仙缘红包 | 持久化存储，未抢完部分过期自动退回 |
| 灵宠 | 每周限购 1 枚兽蛋，兽栏固定 3 格，同时仅可携带 1 只 |
| 灵眼 | 基础 6 个，每 5 名玩家增加 1 个，上限 15 个 |

## Web 预览

当前项目提供只读 Web 预览层，既支持命令行独立启动，也支持随插件自动启动。

### 1. 独立启动

```bash
python web_preview_server.py --db xiuxian_data_lite.db
```

### 2. 插件内置启动

在 `config/game_config.json` 中设置：

```json
{
  "web_server": {
    "enabled": true,
    "auth_enabled": false,
    "guest_access": true,
    "host": "0.0.0.0",
    "port": 8765
  }
}
```

| 配置项 | 说明 |
|---|---|
| `enabled` | 为 `true` 时，插件加载后自动启动只读 Web 预览 |
| `auth_enabled` | 是否启用 Web 轻量鉴权预留层，当前仅提供状态接口，不拦截现有只读页面 |
| `guest_access` | 是否允许游客访问当前只读页面，默认开启 |
| `host` | 监听地址 |
| `port` | 监听端口 |

当 `enabled` 为 `false` 时，仍可继续使用命令行脚本单独启动。
