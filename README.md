# AstrBot 模拟修仙 v2

> 版本：`v3.2.24`  
> 许可证：`AGPL-3.0`  
> 作者：`xiaojuwa`

这是一个运行在 AstrBot 中的文字修仙插件，包含修炼、突破、装备、世界 Boss、秘境、历练、宗门、洞天、灵田、灵眼、灵宠、银行、黑市，以及配套的 Web 预览页面。

## 功能概览

| 模块 | 说明 |
|---|---|
| 基础修仙 | 创建角色、查看信息、签到、改道号、弃道重修、逆天改命 |
| 修炼突破 | 闭关、出关、突破、轻量天劫、悟道、福缘 |
| 装备与背包 | 装备、丹药背包、储物戒、搜索、赠予 |
| 商店与银行 | 丹阁、器阁、百宝阁、银行、普通贷款、突破贷款、黑市 |
| 战斗玩法 | 切磋、决斗、世界 Boss、传承挑战 |
| 探索玩法 | 历练、秘境、悬赏、炼丹 |
| 长线玩法 | 宗门、洞天、灵田、双修、灵眼、灵宠 |
| 社交互动 | 送灵石、论道、仙缘红包 |
| Web 预览 | 角色、排行榜、Boss、秘境、洞天、灵田、灵眼、银行等只读页面 |

## 常用指令

### 入门基础

| 指令 | 说明 |
|---|---|
| `我要修仙 [灵修/体修]` | 创建角色 |
| `我的信息` | 查看角色面板 |
| `签到` | 领取每日奖励 |
| `改道号 <名称>` | 修改道号 |
| `弃道重修 [确认]` | 删除角色并进入冷却 |
| `逆天改命` | 花费 10000 灵石重置灵根 |

### 修炼突破

| 指令 | 说明 |
|---|---|
| `闭关` | 开始修炼 |
| `出关` | 结算修为，并可能触发悟道 |
| `突破信息` | 查看突破要求、成功率与天劫提示 |
| `突破 [丹药名]` | 尝试突破 |
| `悟道信息` | 查看悟道说明 |

### 商店与银行

| 指令 | 说明 |
|---|---|
| `银行` | 查看存款、贷款和利息 |

## 当前规则摘要

| 模块 | 规则 |
|---|---|
| 世界 Boss | 每日可挑战 4 次，次日重置 |
| 切磋 | 直接开始，仅用于战力评估，不消耗气血/灵气 |
| 决斗 | 改为请求制，对方接受后才会正式开打 |
| 论道 | 双方各获得少量修为，并进入 1 小时冷却 |
| 送灵石 | 单次最低 `100`，单次最高 `50000`，每日上限 `200000` |
| 仙缘红包 | 持久化红包，过期未领完自动退款 |
| 灵宠 | 每周限购 1 枚兽蛋，兽栏固定 3 格，同时只能携带 1 只 |
| 灵眼 | 基础 6 个，每 5 名玩家增加 1 个，上限 15 个 |

## Web 预览

项目内提供独立的只读 Web 预览层，默认启动方式：

```bash
python web_preview_server.py --db xiuxian_data_lite.db
```

默认监听配置可在 `config/game_config.json` 中修改：

```json
{
  "web_server": {
    "host": "0.0.0.0",
    "port": 8765
  }
}
```
