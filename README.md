# AstrBot 模拟修仙 v2

> 版本：`v3.1.66`  
> 许可证：`AGPL-3.0`  
> 作者：`xiaojuwa`

一个运行在 AstrBot 中的文字修仙插件，包含修炼、突破、装备、Boss、秘境、历练、宗门、洞天、灵田、灵眼、银行、黑市，以及配套的 Web 预览界面。

## 功能概览

| 模块 | 说明 |
|---|---|
| 基础修仙 | 创建角色、查看信息、签到、改道号、弃道重修、逆天改命 |
| 修炼突破 | 闭关、出关、突破、轻量天劫、悟道、福缘 |
| 装备与背包 | 装备、卸下、丹药背包、储物戒、搜索、赠予、丢弃 |
| 商店与经济 | 丹阁、器阁、百宝阁、银行、突破贷款、黑市 |
| 战斗玩法 | 切磋、决斗、世界 Boss、传承挑战 |
| 探索玩法 | 历练、秘境、悬赏、炼丹 |
| 长线玩法 | 宗门、洞天、灵田、双修、灵眼 |
| Web 预览 | 角色、排行榜、Boss、秘境、洞天、灵田、灵眼、银行等只读页面 |

## 当前规则摘要

| 系统 | 规则 |
|---|---|
| 世界 Boss | Boss 血量低于 30% 时进入 5 回合狂暴，随机触发回血、暴怒、石化、闪避、咆哮之一 |
| Boss 战败 | 玩家战斗 HP 降至 1，5 分钟后可继续挑战；战斗 HP 按时间恢复 |
| 黑市 | 分类定价、分类限购，继续复用现有购买记录表 |
| 轻量天劫 | 金丹及以上突破成功后追加一次天劫判定；失败不会额外死亡 |
| 洞天 | 已支持置换，旧洞天按原价 60% 折算 |
| 储物戒 | 获取物品自动入戒或进入对应背包；不再支持手动存入与取出，玩家侧保留丢弃语义 |

## Web 预览

项目包含一个只读 Web 预览层，可查看玩家总览、排行榜、Boss、秘境、背包、洞天、灵田、灵眼、双修、悬赏、历练和银行等内容。

### 启动方式

```bash
python web_preview_server.py --db xiuxian_data_lite.db
```

### Web 监听配置

在 `config/game_config.json` 中可配置：

```json
{
  "web_server": {
    "host": "0.0.0.0",
    "port": 8765
  }
}
```
